### 1. Clone the Repository
```bash
git clone https://github.com/Inshiya1904/LeaderBoard.git

```

### 2. Install Client Dependencies
```bash
cd Frontend

```

### 4. Install Server Dependencies
```bash
cd Backend
npm install
```

### 5. Start Servers
```bash
# Start Backend
cd Backend
npm run dev
```
